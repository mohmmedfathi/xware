## 📂 Contents
The repository consists of several projects and assignments:

### 🏥 HMS (Hospital Management System)
A simple backend system designed to manage hospital operations.
- **Features:**
  - 🏥 Patient registration and records management.
  - 📅 Appointment scheduling.
  - 👨‍⚕️ Doctor and staff management.
- **Technologies:** Python, SQL.

### ✅ Todo App
A basic task management application.
- **Features:**
  - ➕ Add, edit, and delete tasks.
  - ✅ Mark tasks as completed.
  - 🔍 Filter tasks by status.
- **Technologies:** Python, SQL, Django.

### 🔢 LeetCode SQL
A collection of SQL problem solutions from LeetCode.
- **Focus:**
  - 📝 Writing efficient SQL queries.
  - 🚀 Understanding database concepts.
  - 🎯 Optimizing queries for better performance.

### 📋 Todo App V2
An improved version of the Todo App with additional features.
- **Features:**
  - 🔑 User authentication.
  - 📅 Task deadlines and priority levels.
- **Technologies:** Python, Django, SQL, HTML.
