# Day :five:
## :beginner: Install Postgresql

```
  sudo apt update

  sudo apt install postgresql postgresql-contrib

  sudo systemctl start postgresql.service

  service posgresql status

  sudo -u postgres psql

  sudo systemctl restart postgresql
```
